# Welcome to Platen.io

This is the landing page for Platen, a project for providing scaffolding of digital tabletop roleplaying tools and other interactive web books. The features of this project are intended to shorten the distance between creators and functional, responsive, excellent digital versions of their games.

You can review the available specimen which displays the features currently available for use in your own works.
